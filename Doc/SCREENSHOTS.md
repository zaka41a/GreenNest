## How to Add Screenshots

1. Take screenshots of the following pages:
   - Landing page
   - Products catalog
   - Shopping cart
   - Checkout page
   - Admin dashboard
   - Admin orders management

2. Save them with these exact names in the `doc/` folder:
   - `landing.png`
   - `products.png`
   - `cart.png`
   - `checkout.png`
   - `admin-dashboard.png`
   - `admin-orders.png`

3. Recommended screenshot dimensions: **1920x1080** or **1440x900**

4. Use tools like:
   - macOS: `Cmd + Shift + 4`
   - Windows: `Windows + Shift + S`
   - Browser extensions: Fireshot, Awesome Screenshot

## Screenshot Checklist

- [ ] Landing page
- [ ] Products catalog

- [ ] Checkout form
- [ ] Admin dashboard with statistics
- [ ] Admin orders table

## Tips for Great Screenshots

- Use sample data with realistic values
- Ensure UI is fully loaded before capturing
- Use consistent browser zoom (100%)
- Capture full page or relevant sections
- Avoid sensitive information (real emails, passwords)
